# 📊 交易复盘系统 (Trade Review System)

一个基于 Web 的个人外汇/期货交易复盘系统，帮助交易者系统性地回顾每一笔交易，持续提升交易水平。

## 功能特性

### 🔍 交易复盘
- 导入 MT4 平台的历史交易记录
- 对每笔交易进行**结构化复盘**，包括：
  - 基本面背景（数据发布、央行政策、地缘事件）
  - 大周期趋势状态（日线/周线级别方向）
  - 小周期下单信号（K线形态、指标信号）
  - 交易理论/系统（道氏/缠论/谐波/裸K等）
  - 入场质量评分、情绪状态记录
  - 经验教训与改进方向
- 标签系统：给复盘打标签，便于后续统计分析

### 📈 数据统计
- **资金曲线图**：账户余额走势可视化
- **月度盈亏**：按月统计交易表现
- **品种分析**：按交易品种统计胜率和盈亏
- **方向分析**：做多 vs 做空表现对比
- **复盘洞察**：高频标签、趋势分布、入场质量、情绪分布
- **复盘效果对比**：有复盘 vs 未复盘交易的表现差异

### 📋 交易计划
- 创建详细的交易计划（基本面+技术面分析）
- 设定入场价、止损、目标位、风险比例
- 执行追踪：记录实际入场/出场/盈亏
- 计划 vs 实际对比
- 执行评分

### 📥 数据导入
- 支持 MT4 标准 CSV 导出格式
- 支持中英文列名自动识别
- 自动去重（按 Ticket 号）
- 提供样例文件下载

## 快速开始

### 安装

```bash
# 1. 进入项目目录
cd trade-journal

# 2. 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动系统
python app.py
```

### 使用

1. 浏览器打开 `http://localhost:5000`
2. 先到「导入数据」页面上传 MT4 导出的 CSV
3. 在「交易记录」中查看导入的订单
4. 点击「复盘」对每笔交易进行结构化回顾
5. 在「统计分析」查看资金曲线和各项指标
6. 在「交易计划」制定和执行交易计划

## 技术栈

- **后端**: Python Flask + SQLAlchemy
- **数据库**: SQLite（零配置，自动创建）
- **前端**: 原生 HTML/CSS/JS + Chart.js
- **部署**: 单文件启动，无需额外服务

## 核心目的

这个系统的设计哲学是：**不追求预测市场的准确性，而是追求交易的纪律性和自我认知**。

通过系统性的复盘，帮助交易者：
1. 发现自己的**交易模式**（什么情况下容易盈利/亏损）
2. 识别**情绪化交易**的触发条件
3. 验证自己的**交易理论**是否经得起检验
4. 将**交易计划**和**实际执行**之间的差距可视化
5. 通过长期积累，建立自己的 **交易数据库**，持续迭代交易系统
